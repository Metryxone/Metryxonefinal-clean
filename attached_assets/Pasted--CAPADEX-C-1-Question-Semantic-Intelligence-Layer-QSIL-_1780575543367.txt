# CAPADEX C-1

## Question Semantic Intelligence Layer (QSIL)

### Objective

Transform CAPADEX from Concern-Level Intelligence to Question-Level Intelligence.

Current State:

Ontology Intelligence = Operational

Bridge Tag Intelligence = Operational

Signal Intelligence = Operational

Metadata Intelligence = Operational

Question Intelligence = Limited

Question Differentiability = Limited

Assessment Intelligence is currently constrained because many questions inherit identical metadata from their parent concern or bridge tag.

The goal of C-1 is to create semantic differentiation at the individual question level.

---

## PHASE 1 — Question Intelligence Audit

Audit all questions across the CAPADEX repository.

For each Concern and Bridge Tag calculate:

* Question Count
* Metadata Uniformity
* Age Diversity
* Persona Diversity
* Stage Diversity
* Behavior Diversity
* Capability Diversity
* Signal Diversity
* Context Diversity
* Archetype Diversity

Measure:

Question Uniformity Score

Question Diversity Score

Question Differentiability Score

Identify:

* Tags with >80% metadata similarity
* Tags with low differentiation
* Tags producing generic assessments
* Tags likely causing relevance drift

Generate:

question_uniformity_report

question_diversity_report

question_differentiability_report

STOP

Display findings before proceeding.
