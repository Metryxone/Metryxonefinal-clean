========================================
RANKING FORMULA
========================================

Implement EXACTLY:

score =
(projected_ei_lift / max(effort_hours, 0.5))
× velocity_multiplier
× confidence_multiplier
× weight_multiplier

velocity_multiplier:
accelerating → 1.4
stabilizing → 1.1
flat → 1.0
declining → 1.6

confidence_multiplier:
A → 1.0
B → 0.9
C → 0.7
D → 0.4

weight_multiplier:
role competency weight
