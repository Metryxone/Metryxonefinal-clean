# WC-8 Track H — World-Class Future Readiness Gap Report

Benchmarks CAPADEX against the capability *categories* of future-facing platforms. (Comparison is
capability-class, not a named-competitor scrape — no external data was fetched for this audit.)

## H.1 Capability benchmark
| Capability class | World-class expectation | CAPADEX today | Gap |
|------------------|-------------------------|---------------|-----|
| Career intelligence | Behavioural + outcome + journey per user | **Met** — WC-3 L1/L2/L3 + decision orchestrator | None |
| Employability scoring | Live, scored, monetized index | **Met** — `employability_index` ready + `employability_readiness` outcome | Minor (reframe to "future") |
| AI career navigation | Occupation exposure + skill-shift guidance | **Missing** — only AI *anxiety* detection (440 Qs), no adaptation | **Large** |
| Mentoring | Decision-matched mentor types | **Met** — mentor-bridge (WC-7B) | None |
| Entrepreneurship readiness | Founder-trait + venture-readiness scoring | **Missing** — 89 context Qs, no outcome/product | **Large** |
| Lifelong learning / future skills | Skill-demand forecast + personalized path | **Partial** — growth-plan bridge live, no skill taxonomy | **Medium** |
| Adaptive growth planning | Decision→plan→mentor→skills | **Mostly met** — chain live except final Future-Skill link | Small |

## H.2 What's missing — the four lists the spec asks for
- **Missing capabilities:** AI adaptation guidance (not just detection); occupation→automation
  exposure; founder-trait scoring; explicit skill-obsolescence/layoff context routing.
- **Missing intelligence layers:** an AI-resilient **skill taxonomy** (the keystone shared asset);
  a future-skill **demand signal**; future-readiness **outcome models** (ai_readiness,
  entrepreneurial_readiness, career_resilience, future_skills).
- **Missing products:** AI Career Navigator, Future Skills Planner, Career Resilience Index,
  Entrepreneurship Index (Employability Index exists and needs only reframing).
- **Missing revenue streams:** premium future-readiness SKUs on the existing CAP_INS/GRW/MAS ladder;
  no recurring "AI readiness" or "future skills" subscription surface today.

## H.3 Where CAPADEX already leads
Behavioural depth (4-tier signal ontology, 30k context-classified questions), an honest
flag-gated activation architecture (WC-7A/B/C), and a **live employability product** put CAPADEX
ahead of pure content/job-board tools. The gap is **forward-looking activation**, not behavioural
intelligence — CAPADEX's core engine is world-class; its future-readiness *surfaces* are not yet.

## H.4 Single biggest lever
Build the **AI-resilient skill taxonomy + occupation-exposure reference once**; it closes the AI,
Future-Skills, and Employability-2.0 gaps simultaneously and unlocks 3 of the 5 missing products.
