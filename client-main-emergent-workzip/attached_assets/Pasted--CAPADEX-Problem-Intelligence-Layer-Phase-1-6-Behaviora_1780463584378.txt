# CAPADEX Problem Intelligence Layer

## Phase 1.6 – Behavioral Intelligence Layer

### Objective

Build the missing behavioral layer between capabilities and problems.

Current CAPADEX Structure:

Capability
↔
Problem State

Target Structure:

Capability
↔
Problem State
↔
Observable Behavior

This phase creates the evidence layer required for explainable assessments, AI coaching, interventions, reporting, and future archetype discovery.

# PHASE 1.6A – BEHAVIOR GENERATION

For every normalized concern generate:

Minimum:
3 behaviors

Maximum:
5 behaviors

Behaviors must be:

* observable
* measurable
* real-world
* age appropriate

Avoid abstract language.

---

Example

Capability:
Time Management

Problem:
Time Management Weakness

Behaviors:

* Frequently misses deadlines
* Starts assignments at the last minute
* Struggles to estimate task duration
* Leaves work incomplete
* Rushes submissions

---

Example

Capability:
Communication Skills

Problem:
Communication Weakness

Behaviors:

* Avoids speaking in group discussions
* Gives very short responses
* Hesitates when presenting ideas
* Rarely asks questions
* Struggles to explain thoughts clearly

# PHASE 1.6G – DUPLICATE BEHAVIOR DETECTION

* semantic duplicates
* wording duplicates
* severity duplicates
