Build a production-grade, enterprise-ready full-stack web application called:

“Metryxone – Gamified Learning & Career Intelligence Platform”

---

TECH STACK (MANDATORY)

Frontend:

* React (Next.js preferred)
* Tailwind CSS
* Zustand or Redux Toolkit

Backend:

* Node.js (NestJS preferred or Express with clean architecture)
* REST APIs (with optional GraphQL layer)

Database:

* PostgreSQL (with Prisma ORM)

Caching & Queues:

* Redis (for leaderboards, sessions, rate limiting)
* BullMQ or similar (background jobs)

Real-time:

* WebSockets (leaderboard updates, notifications)

DevOps:

* Dockerized setup
* Environment-based config
* Logging (Winston)
* API rate limiting

---

CORE OBJECTIVE

Build a scalable platform where students:

* Log in daily and earn rewards
* Complete personalized missions
* Earn XP, Coins, and Levels
* Maintain streaks
* Improve measurable skills
* Participate in teams and competitions
* Interact with mentors
* Redeem rewards (digital + physical + career opportunities)

---

USER ROLES

* Student
* Parent (linked to student)
* Mentor
* Admin
* Vendor (reward partner)
