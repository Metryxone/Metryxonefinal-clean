Create a premium, production-ready chatbot UI for "MetryxOne AI Assistant" using React (Vite preferred) and Tailwind CSS.

The design must combine:

* Glassmorphism UI
* Dark/Light mode toggle
* ChatGPT-style typing animation
* WhatsApp-style chat layout
* A consistent branded design system

The final result should feel like a polished SaaS product (similar to ChatGPT + WhatsApp + Apple design quality).

---

# 🎨 DESIGN SYSTEM (MetryxOne Branding)

## Colors

* Primary: #2563eb (blue)
* Accent: #22c55e (green for success/health)
* Background Light: #f9fafb
* Background Dark: #0f172a
* Glass Background: rgba(255,255,255,0.1) / rgba(15,23,42,0.4)
* Borders: rgba(255,255,255,0.2)

## Typography

* Font: Inter (fallback: system-ui)
* Headings: semi-bold
* Body: regular, clean spacing

## Feel

* Calm, intelligent, premium
* Not flashy, but visually rich
